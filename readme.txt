WPLMS Unit Access Addon adds functionality to set number of times a user can see content of unit.

== Description ==

This plugin adds functionality to set number of times a user can see content of unit


Tutorial On how to setup and get started : <a href="https://vibethemes.com/documentation/wplms/knowledge-base/limit-unit-views-in-courses/">link</a>

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'WPLMS Unit Access Addon'
3. Activate WPLMS Unit Access Addon from your Plugins page. 

= From WordPress.org =

1. Download WPLMS Unit Access Addon.
2. Upload the 'wplms-unit-access-addon' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate WPLMS Unit Access Addon from your Plugins page. 
= Once Activated =

1. After installing this plugin you will see a setting for number of times user can view the unit in unit settings .
2. User can only view/load unit the number of times you ste in unit settings .
